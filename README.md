# MatrixIslandsCounter

This will count how many islands are in a sliced matrix.
Use a symbol ('9' in this example) to know the limits of the islands

matrix(5,5)

0 0 0 0 0<br />
0 0 0 0 0<br />
0 0 0 0 0<br />
0 0 0 0 0<br />

so

0 0 9 0 0<br />
0 0 9 0 0<br />
0 0 9 0 0<br />
0 0 9 0 0<br />

then

1 1 9 2 2<br />
1 1 9 2 2<br />
1 1 9 2 2<br />
1 1 9 2 2<br />

total island = 2
